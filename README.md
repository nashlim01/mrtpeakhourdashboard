# MRT Putrajaya Line Strategy Simulator

A React-based dashboard for simulating and comparing operational strategies during CBD disruption scenarios on the MRT Putrajaya Line.

## 🚀 Quick Start

### Option 1: Local Server (Recommended)
```bash
# Navigate to project folder
cd mrtpeakhourdashboard/

# Python 3
python3 -m http.server 8000

# Or Node.js
npx serve .

# Open in browser
http://localhost:8000 OR your dedicated port if you're using Node.js
